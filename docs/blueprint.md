# **App Name**: Bagga Bugs Enhanced

## Core Features:

- Email Sending: Send personalized emails to recipients listed in a Google Sheet without daily limits.
- Email Formatting: Format and clean email addresses to prevent spam and ensure inbox delivery.
- Content Personalization: Customize email subject and body with personalized content using merge tags.
- Automatic Open Tracking: Track email opens automatically by logging open events in a dedicated tracking log.
- Campaign Management UI: Provide a user interface within Google Sheets to configure and manage email campaigns.
- Assistant: Show a webapp page of clear instructions of how to use the system.

## Style Guidelines:

- Primary color: Deep Indigo (#6666AA) for a professional and reliable feel.
- Background color: Very light gray (#FAFAFC), almost white, providing a clean backdrop.
- Accent color: Violet (#9966BB) for highlights and call-to-action elements.
- Body and headline font: 'Inter', a sans-serif, for clear and modern readability. Note: currently only Google Fonts are supported.
- Use simple, clean icons for key actions such as send, track, and settings.
- Maintain a clear and consistent layout across the Google Sheet and any UI elements to aid usability.
- Use subtle animations, such as progress bars, when sending emails to provide visual feedback.
